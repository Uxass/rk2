Frontend

Frontend - это та область веб-разработки, которая занимается созданием пользовательского интерфейса сайта или приложения, с которым взаимодействует пользователь. Frontend разработчик занимается написанием кода на языках программирования HTML, CSS и JavaScript, который отвечает за визуальное и функциональное поведение интерфейса.
